# Todo App - Currently Running

## Servers Status
✅ **Backend API**: Running on `http://localhost:8080`
✅ **Frontend App**: Running on `http://localhost:3003`

## How to Access
1. **Frontend Application**: Open your browser and go to [http://localhost:3003](http://localhost:3003)
2. **API Documentation**: Visit [http://localhost:8080/docs](http://localhost:8080/docs) for API documentation
3. **API Health Check**: [http://localhost:8080/](http://localhost:8080/) returns API status

## Application Features
- User Registration and Login
- Task Management (Create, Read, Update, Delete)
- Task Filtering (All, Active, Completed)
- Priority and Due Date Management
- Responsive UI with Modern Design

## Next Steps
1. Open your browser to `http://localhost:3003`
2. Register a new account or sign in if you have one
3. Start managing your tasks!

## Stopping the Application
To stop the servers, press `Ctrl+C` in each terminal window where they're running.

## Troubleshooting
- If you close these terminals, the servers will stop
- To restart, run: `start-project.bat` or manually start backend and frontend
- If ports change, check the terminal output for the correct URLs
