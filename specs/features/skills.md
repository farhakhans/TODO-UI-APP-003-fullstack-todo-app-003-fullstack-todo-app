# Core Application Skills

This document defines the five core skills of the application that users can immediately understand before signup or login, highlighting the product's functional value.

## Task Management Skill

The Task Management Skill enables users to efficiently organize and track their work through a comprehensive task management system. Users can:

- **Create tasks** with detailed descriptions, due dates, and priority levels
- **Update tasks** as requirements change or additional information becomes available
- **Delete tasks** that are no longer relevant or needed
- **Complete tasks** to mark them as finished and track progress
- **Persistent storage** ensures all tasks remain accessible across sessions, devices, and application updates

This skill provides the fundamental value proposition of the application, allowing users to maintain control over their responsibilities and improve productivity through structured task organization.

## Multi-User Isolation Skill

The Multi-User Isolation Skill ensures complete data separation between users while enabling a shared platform experience. This skill provides:

- **Individual task visibility** where each user sees only their own tasks, maintaining privacy and focus
- **Secure ownership enforcement** that prevents unauthorized access to or modification of another user's tasks
- **Personalized workspace** that adapts to each user's specific organizational preferences and workflow
- **Scalable user management** that supports unlimited users without cross-contamination of data

This skill enables the application to serve multiple users simultaneously while maintaining the security and privacy essential for personal and professional task management.

## Secure Authentication Skill

The Secure Authentication Skill provides robust user identity management and session security. This skill encompasses:

- **Signup capability** allowing new users to create accounts with secure credential storage
- **Signin functionality** providing secure access to user-specific data using Better Auth technology
- **JWT-based session security** ensuring authenticated sessions remain secure and valid
- **Password recovery** for account access restoration when needed
- **Secure credential handling** protecting sensitive authentication information

This skill establishes the trust foundation necessary for users to confidently store their tasks and personal information in the application.

## Responsive Web Experience Skill

The Responsive Web Experience Skill ensures optimal usability across all device types and screen sizes. This skill delivers:

- **Mobile compatibility** providing full functionality on smartphones with touch-optimized interfaces
- **Tablet optimization** offering enhanced layouts that take advantage of medium-sized screens
- **Desktop experience** delivering comprehensive functionality on larger screens with keyboard navigation
- **Modern, professional UI** featuring intuitive design patterns and visual appeal
- **Consistent experience** ensuring functionality and appearance remain uniform across all platforms

This skill ensures users can access their tasks anytime, anywhere, regardless of their preferred device.

## Scalable Architecture Skill

The Scalable Architecture Skill provides the technical foundation for reliable, high-performance operation. This skill includes:

- **Frontend and backend separation** enabling independent development, deployment, and scaling of client and server components
- **Cloud-ready database** utilizing Neon PostgreSQL for high availability, performance, and automatic scaling
- **Performance optimization** ensuring fast response times regardless of user load or data volume
- **Reliability and uptime** maintaining consistent availability for users worldwide
- **Future growth support** accommodating increasing numbers of users and data without performance degradation

This skill ensures the application can grow with user needs while maintaining consistent performance and reliability.
